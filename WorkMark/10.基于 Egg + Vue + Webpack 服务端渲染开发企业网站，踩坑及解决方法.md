### 一、安装egg

egg 需要 Node.js 8.x以上版本。  

在终端输入以下命令全局安装egg:  
> npm i egg-init -g 

安装手脚架： 
> npm install easywebpack-cli -g  

命令行运行：  
> easywebpack init
